# TEST_CREATE

## 目的
- 今回の中間/期末テストを作る

## 入力
- テスト名
- テスト開始日
- テスト終了日
- 日程ごとの科目

## 画面ルール
- 複数日程を追加できる
- 1日ごとに科目を選べる
- 入力後に `前回テストを引き継ぐ` へ進む

## エラー状態
| 場面 | 挙動 |
|---|---|
| テスト名が空のまま次へ | 「テストに名前をつけてから進めましょう」をインライン表示 |
| 終了日が開始日より前 | 入力を開始日以降に制限。エラー文は出さず、ヒント文「開始日以降の日付が選べます」を常時表示 |
| 科目が0件のまま次へ | 「科目を1つ追加してから進めましょう」をインライン表示 |

## 注意点
- 時限や試験時間までは持たない
- 科目単位で止める
