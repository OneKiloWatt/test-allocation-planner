# TERMS

## 目的
- サービスの利用条件をユーザーに明示する

## 配置
- `/terms` として独立ページ
- アカウントメニューから遷移
- ヘッダーに「戻る」

## UIルール
- 本文は読みやすいフォントサイズで縦スクロール
- セクションごとに見出しを立てる
- 難しい表現は平易な言葉に言い換える（中高生が読める水準）
- 最終更新日を上部に明示する

## やらないこと
- メニュー内への本文展開
- モーダルでの表示

## 備考
- 法的文章の本文は別途法務確認が必要
- 特に「未成年者同意の取得方法」「免責の限定責任文言」「管轄条項」は公開前に要確認
