# TARGET_SCORE

## 目的
- 今回テストの教科別目標点数を決める
- 前回データを参考にしながら目標を設定する（CARRY_OVERを統合）

## 入力
- 教科別目標点数

## 表示（科目ごとの行）

```
数学
目標点: [___] 点
▼ 前回を見る → 72点 / 180分
```

- 前回データは折りたたみ表示（デフォルトは閉じた状態）
- 前回データがない科目（初回・新科目）は折りたたみを出さない
- 前回データがある科目のみ `▼ 前回を見る` を表示

## 科目名が変わる場合の対処
- 科目名ミスマッチが1件以上ある場合、画面上部にバナーを1回だけ表示する

```
「前回と科目名が変わったものがあります → 確認する」
```

- 「確認する」をタップすると、該当科目に `前回の○○と同じ？` チップが出る
- チップで選択するか `前回データなしで始める` を選ぶ
- 全科目が同名なら、バナーは出さない

## UI方針
- ここでは `目標をどこに置くか` だけ考えさせる
- 前回データは参考であり、強制ではない
- 日程や部活設定は混ぜない

## エラー状態
| 場面 | 挙動 |
|---|---|
| 点数が100超 | 「100点までの数字を入れてみましょう」をインライン表示 |
| 全科目未入力のまま次へ | 「1科目だけでも目標を入れると、配分の計算ができます」をインライン表示 |
