# DAILY_PLAN

## 目的
- 日付ごとに、何の科目をどのくらいやるかを確認または調整する

## 表示
- 上部:
  - 教科別のおすすめ配分サマリー
- 中部:
  - 日付一覧、または時間のないカレンダー
- 日付
- 科目
- 目安時間

例:
- 数学 8時間
- 9/10 数学 45分
- 9/11 英語 30分 / 理科 30分
- 9/12 数学 30分

## 自動モードの導線
- `PLAN_MODE` で設定完了後に入る
- 上部におすすめ配分サマリーが表示される
- 自動生成された日ごとのプランを確認する
- 必要なら軽く修正する

## 手動モードの導線
- `PLAN_MODE` で `手動で組む` を押したら即入る
- 上部におすすめ配分サマリーが表示される
- 日付一覧をタップすると、その日の科目と目安時間を追加/編集できる

## 手動モード未入力時
- まだどの日にも学習プランを置いていない状態を持つ
- このときは空状態UIを出す

表示例:
- `まだ学習プランがありません`
- `日付を押して、やる科目と目安時間を置いていきましょう`
- `最初の1日を追加する`

主操作:
- 日付をタップして追加
- 既に入力済みの日付をタップして編集

## 重要な意味
- この画面で `教科ごとの総量` と `日ごとの割り当て` をまとめて可視化する
- `次へ` を押した後に、今回の全体像が一気に見えることを重視する
- `残り時間` は初回表示では出さず、進捗登録後に返す

## やらないこと
- 時刻の入力
- Googleカレンダーのような時間単位編集
- 複雑なドラッグ前提UI
