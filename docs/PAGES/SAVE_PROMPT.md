# SAVE_PROMPT

## 目的
- 匿名利用中に、記録がこの端末に依存することを軽く知らせる
- ログインへの導線を、強制ではなく提案として出す

## 出すタイミング
- テスト作成後
- 学習プラン保存後
- 進捗記録後
- 結果入力後

## 表示内容
- `この記録は現在この端末に保存されています`
- `ブラウザや端末を変えると見られなくなる場合があります`
- `残したい場合はログインしてください`

## ボタン
- `あとで`
- `ログインする`

## 文言ルール
- 脅しすぎない
- 制約は明確に伝える

使う:
- `別の端末でも使いたい場合はログインしてください`
- `この端末に保存されています`

使わない:
- `今すぐログインしないと危険`
- `必ずログインしてください`
