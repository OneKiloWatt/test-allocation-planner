# PLAN_MODE

## 目的
- おすすめ配分を、実際の日程へどう落とすかを決める

## 前提
- この画面の前に `目標点数入力` を終えている
- ここでは `いつ勉強できるか` と `どう落とすか` だけに集中させる

## 最初に選ばせるもの
- `自動で組む`
- `手動で組む`

## 自動で組む
### 主入力
- 平日部活ありの日の勉強可能時間
- 平日部活なしの日の勉強可能時間
- 土曜の勉強可能時間
- 日曜の勉強可能時間
- 部活がある曜日

### 詳細設定
- テスト1週間前は部活なし扱い

### UI方針
- 主入力は最初から見せる
- `こだわって設定する` を押すと詳細設定が開く
- 設定完了後に `日ごとの学習プラン` 画面へ進む

### MVPでやらないこと
- 例外日の細かい上書き
- 特殊ケースの複雑なパターン入力

## 手動で組む
### できること
- 日付ごとに科目を置く
- その日の科目と時間を編集する

### やらないこと
- Googleカレンダーのような時間単位の予定編集
- ドラッグ前提の複雑なUI

### UI方針
- 最初に生活パターン入力を要求しない
- `手動で組む` を押したら、すぐ `日ごとの学習プラン` 画面へ進む
- ベースは `時間のないカレンダー` または `日付一覧`
- 主操作は `日付を押して追加/編集`
- `+` は補助としてあってもよいが、主操作にしない
- まずは日ごとの学習テーマを置けるだけでよい
