# PAGES

画面仕様の一覧。実際の利用フローに沿って読めるようにまとめています。

## 基本フロー

1. [TOP.md](TOP.md) - 匿名利用とログインの入口
2. [AUTH.md](AUTH.md) - ログイン、新規登録
3. [HOME.md](HOME.md) - 状態別のメイン導線
4. [TEST_CREATE.md](TEST_CREATE.md) - テスト日程と科目の登録
5. [TARGET_SCORE.md](TARGET_SCORE.md) - 教科ごとの目標点入力
6. [PLAN_MODE.md](PLAN_MODE.md) - 自動作成か手動作成かの選択
7. [ALLOCATION_RESULT.md](ALLOCATION_RESULT.md) - 推奨配分サマリー
8. [DAILY_PLAN.md](DAILY_PLAN.md) - 日ごとの学習プラン
9. [PROGRESS_LOG.md](PROGRESS_LOG.md) - 学習実績の記録
10. [SAVE_PROMPT.md](SAVE_PROMPT.md) - 匿名利用時の保存案内
11. [RESULT_ENTRY.md](RESULT_ENTRY.md) - テスト結果の入力
12. [REVIEW.md](REVIEW.md) - テスト後の振り返り

## 補助仕様

| ファイル | 用途 |
|---|---|
| [TIPS.md](TIPS.md) | 画面ごとに出す短い学習豆知識 |
| [ACCOUNT_MENU.md](ACCOUNT_MENU.md) | ヘッダー右上アカウントメニュー（ログアウト・退会・利用規約・プライバシーポリシー） |
| [TERMS.md](TERMS.md) | 利用規約ページ |
| [PRIVACY.md](PRIVACY.md) | プライバシーポリシーページ |

## 補足

- `ALLOCATION_RESULT` は独立画面ではなく `DAILY_PLAN` 上部サマリー
- 旧 `CARRY_OVER` は `TARGET_SCORE` に統合済み
- 廃止済み仕様は [`../ARCHIVE/README.md`](../ARCHIVE/README.md) に移動
