# TIPS

## 方針
- アプリ上の表示テキストは出典なし・占いくらいの温度感
- 内部メモとして根拠論文を残す（ポートフォリオ用）
- 各画面でローテーション表示（日付ベースのインデックスで切り替え）
- 3行以上は書かない

## HOME（10個）

**[1]** 一気にやるより、少し時間をあけて復習するほうが、ちゃんと頭に残るよ。今日やったことは明日もちょっとだけ見直してみて。
*(内部メモ: 分散学習 / Ebbinghaus 1885)*

**[2]** 教科書を読み返すより、「さっき何を学んだっけ？」と自分に問いかけるほうが、記憶により残りやすいんだって。ノートを閉じてテストしてみよう。
*(内部メモ: 想起練習 / Roediger & Karpicke 2006)*

**[3]** 睡眠中に脳は今日の記憶を整理してくれてる。ちゃんと寝ることも、勉強の一部だよ。
*(内部メモ: 睡眠と記憶 / Stickgold 2005)*

**[4]** 25分集中して5分休む、それだけで脳がリフレッシュされる。タイマーをセットしてみて。
*(内部メモ: ポモドーロ / Cirillo)*

**[5]** 脳が元気なうちに、いちばん手ごわい科目からやっつけよう。あとがラクになるよ。
*(内部メモ: 難しい科目は最初に / Bjork 1994)*

**[6]** 「なんでそうなるんだろう？」って自分に問いかけながら読むと、ただ読むよりずっと頭に入ってくるよ。
*(内部メモ: 精緻化質問 / Woloshyn 1992)*

**[7]** 「がんばる」より「この問題を3問解く」って決めたほうが、動きやすくなる。小さく具体的に決めてみて。
*(内部メモ: 目標設定 / Locke & Latham 2002)*

**[8]** 歌詞ありの音楽は集中の邪魔になりやすい。勉強中はインストにしてみて。
*(内部メモ: 歌詞なし音楽 / Perham & Currie 2014)*

**[9]** 誰かに説明しようとすると、自分のわかってないところが見えてくる。友達に教えるつもりで声に出してみよう。
*(内部メモ: 人に教える / Nestojko 2014)*

**[10]** 間違えた問題こそ、いちばんの先生。なぜ違ったかを考えると、そこから一気に伸びられるよ。
*(内部メモ: エラーベース学習 / Metcalfe 2017)*

## PROGRESS_LOG（10個・保存後に表示）

**[1]** 勉強したあとに少しボーッとする時間、実は脳が記憶を定着させてる大事な時間なんだよ。休憩もちゃんとサボらずとってね。
*(内部メモ: 勉強後の休息 / Dehaene 2021)*

**[2]** 夜に覚えたことは、寝てる間に脳がしっかり整理してくれる。今夜もちゃんと寝てね。
*(内部メモ: 睡眠と記憶 / Stickgold 2005)*

**[3]** 体を動かすと、脳への血流もよくなって記憶の定着を助けてくれる。軽く散歩するだけでも効果あるよ。
*(内部メモ: 運動と記憶 / van Dongen 2016)*

**[4]** 「できた！」って感じた瞬間、脳はすごく喜んでる。その気持ち、大事にしてね。
*(内部メモ: ドーパミンと達成感 / Schultz 1998)*

**[5]** 勉強したすぐあとにスマホを見ると、せっかくの記憶が上書きされやすくなっちゃう。少しだけ待ってみて。
*(内部メモ: 勉強後スマホ / Tambini 2010)*

**[6]** 一度やっただけじゃ忘れるのは当たり前。少し間をあけてまた見直すと、ぐっと定着するよ。
*(内部メモ: 間隔をあけた復習 / Ebbinghaus)*

**[7]** 水分が足りないと集中力がにぶくなりやすいって知ってた？勉強前にコップ一杯、飲んでみて。
*(内部メモ: 水分補給 / Edmonds 2013)*

**[8]** 「自分にはできる」って気持ちは、ほんとうに実力を引き出してくれる。まず一問、やってみよう。
*(内部メモ: 自己効力感 / Bandura 1997)*

**[9]** 簡単すぎる問題ばかりやっても伸びにくい。少し難しいと感じるくらいのほうが、実は力がついてるサインだよ。
*(内部メモ: 困難な問題ほど深い / Bjork 1994)*

**[10]** 誰かに説明してみると、自分の理解があいまいなところがはっきりわかる。口で言えたら、本当にわかってる証拠だよ。
*(内部メモ: 人に説明 / Koh 2018)*

## DAILY_PLAN（10個）

**[1]** 同じ科目だけを続けるより、いくつかの科目をまぜながら勉強するほうが、頭の使い方が鍛えられるよ。ちょっと試してみて。
*(内部メモ: インターリービング / Kornell & Bjork 2008)*

**[2]** 今日だけ詰め込むより、少しずつ日をあけて復習するほうが、ちゃんと記憶に残りやすいよ。今日の分は明日も少し見直してね。
*(内部メモ: 分散学習 / Cepeda 2006)*

**[3]** 脳が元気なうちに、いちばん苦手な科目を持ってきてみて。意外とサクサク進むかもしれないよ。
*(内部メモ: 難しい科目は最初に / Bjork 1994)*

**[4]** 読むだけじゃなくて、自分でテスト形式で問いかけてみると、ずっと効果的に覚えられるよ。
*(内部メモ: テスト形式の練習 / Roediger & Karpicke 2006)*

**[5]** 集中が切れてきたら、少し休憩してみて。脳がリセットされて、また動き出しやすくなるよ。
*(内部メモ: 休憩と記憶定着 / Dewar 2012)*

**[6]** 「今日は何をするか」を前もって書き出しておくと、勉強をスタートしやすくなるよ。朝に一度やってみて。
*(内部メモ: 計画を書き出す / Gollwitzer 1999)*

**[7]** 最初は解けそうな問題から手をつけると、気持ちが乗ってくる。勢いをつけてから難しいのに挑もう。
*(内部メモ: 簡単な問題から / Bandura)*

**[8]** いつもと違う場所で勉強してみると、気分が変わって集中しやすくなることもあるよ。図書館も試してみて。
*(内部メモ: 場所を変える / Smith 1978)*

**[9]** 同じ科目をまとめてやるんじゃなくて、いくつかをまぜるスケジュールにしてみると、実力がつきやすいよ。
*(内部メモ: ミックス型スケジュール / Taylor & Rohrer 2010)*

**[10]** 寝る前に今日やったことをさっと見直すと、眠ってる間に脳がそれを整理してくれるよ。おやすみ前の5分、試してみて。
*(内部メモ: 寝る前の復習 / Stickgold 2005)*

## RESULT_ENTRY（10個・保存後に表示）

**[1]** 自分の勉強を記録するだけで、何がうまくいってるかが見えてくる。続けることで自分の成長も感じやすくなるよ。
*(内部メモ: 記録の価値 / Zimmerman)*

**[2]** 間違えた問題をそのままにしないで、「なぜ違ったか」を一言でいいから書いてみよう。それが次の正解につながるよ。
*(内部メモ: エラー分析 / Metcalfe 2017)*

**[3]** 今日できなかったことより、先週できなかったことが今日できてるかに目を向けてみて。ちゃんと伸びてるよ。
*(内部メモ: 成長に注目 / Dweck)*

**[4]** 振り返りをする習慣があると、ただこなすだけより学びが深まりやすいよ。今日の勉強、一言でまとめてみて。
*(内部メモ: 振り返りの価値 / Zimmerman)*

**[5]** 「これ解けるかな」と予測してから問題を解くと、ズレに気づいてより深く学べるよ。自分の理解度を試してみて。
*(内部メモ: 予測とのズレ / Flavell)*

**[6]** 同じようなミスが続いてたら、それはパターンのサイン。どこでつまずいてるか、一度整理してみよう。
*(内部メモ: ミスのパターン / Metcalfe 2017)*

**[7]** 今日いちばん難しかった問題、もう一度だけ見直してみて。そこに伸びるヒントが隠れてるよ。
*(内部メモ: 難しかった問題 / Bjork)*

**[8]** 勉強の記録をつけ続けると、自分の習慣が見えてきて、自然と改善できるようになるよ。小さくてもいいから続けてみて。
*(内部メモ: 記録習慣 / Zimmerman & Schunk)*

**[9]** 勉強中に感じた気持ちを少し振り返ってみると、次にどう取り組むかのヒントが見えてくることがあるよ。
*(内部メモ: 気持ちを振り返る / Pekrun 2011)*

**[10]** ミスをしたとき、脳はそこから学ぼうとフル回転してる。間違えることを怖がらないで、どんどん試してみよう。
*(内部メモ: ミスから学ぶ脳 / Moser 2011)*
