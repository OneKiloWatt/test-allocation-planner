# SECURITY

## 基本方針
- 未成年の学習データを扱う前提で、安全側に倒す
- 取らなくていいデータは取らない
- 断定評価、比較、共有を初期状態でやらない

## 扱うデータ
- テスト名
- テスト日程
- 教科
- 教科別目標点数
- 教科別点数
- 教科別勉強時間
- ブロック日

## 持たないデータ
- 学校名
- 住所
- 順位
- 友人情報
- 保護者情報
- 他人比較用データ

## 危険な仕様
- `このままだと危険`
- `この配分なら80点取れる`
- 他ユーザー比較
- ランキング
- 保護者/学校/塾への自動共有
- 広告目的の外部送信

## MVP時点の安全要件
- 通信は暗号化
- 認証ありの場合は最低限のアクセス制御
- データ削除導線を持つ
- 利用目的を明示する
- 提案文言は `参考プラン` `おすすめ配分` に留める

## 外部依存・外部通信の制約
- 共通基盤では CDN フォント、外部アイコン配信、外部解析タグを禁止する
- 具体的には Google Fonts の `<link>` 読み込み、Font Awesome などの外部 CDN、Google Analytics / GTM / Meta Pixel などの計測タグを MVP の標準構成に入れない
- フォントは `next/font` による自己ホストを原則とし、ビルド成果物に同梱されたものだけを使う
- アイコンはリポジトリに含めるコンポーネント、または npm 依存として監査可能なものだけを使う
- 外部 API / 外部 SaaS への通信は、仕様書で目的・送信データ・失敗時挙動を定義し、Security 確認を通したものに限る

## npm 依存追加ルール
- 依存追加は allowlist 方針で運用し、Next.js / React / TypeScript / Tailwind / shadcn-ui 系など、用途が明確で既存方針に合うものを優先する
- 新規依存は「なぜ標準機能や既存依存では足りないか」を説明できる場合のみ追加する
- `package-lock.json` または利用中 lockfile の差分レビューを必須とし、想定外の transitive dependency 増加を確認する
- メンテ停止、未成熟、難読化済み配布物、インストール時スクリプト依存が強いパッケージは避ける
- 外部通信、認証、暗号、HTML 生成、ファイル処理に関わる依存追加は Security レビュー必須とする

## データ保持方針
- テスト記録はユーザーが削除可能
- 退会または初期化で削除可能
- 長期保持を前提にしない

## 表現ルール
- 使う:
  - `前回より数学の配分を増やす案`
  - `目安`
  - `参考プラン`
- 使わない:
  - `勉強不足`
  - `志望校は無理`
  - `平均より低い`

## 共通入力ポリシー

### 文字数上限
- テスト名: 40文字以内
- 科目名: 20文字以内
- 任意メモ: 200文字以内
- 数値入力欄に付随する自由記述ラベルや補助名: 20文字以内
- 将来フィールドを追加する場合も、短文ラベル系は 40文字以内、自由記述メモ系は 200文字以内を初期基準にし、仕様書へ明記してから拡張する

### 正規化方針
- 保存前に全文字種別共通で Unicode 正規化は行わず、入力値の意味を変える変換は避ける
- テキスト入力は保存前に改行を `\n` へ正規化する
- 単一行入力（テスト名・科目名など）は保存前に先頭末尾の空白を trim する
- 複数行入力（任意メモなど）は保存前に行ごとの内容は維持しつつ、先頭と末尾の連続改行・空白のみ trim する
- 単一行入力に含まれる改行は受け付けず、バリデーションエラーとして扱う

### 表示時エスケープ方針
- ユーザー入力テキストは常にテキストノードとして描画し、HTML として解釈しない
- React の標準エスケープを前提とし、`dangerouslySetInnerHTML` は MVP では使用しない
- 改行表示が必要なメモは、エスケープ済みテキストを CSS の `white-space` 制御で見せる

### 共通エラーメッセージ方針
- 文言トーンは中高生向けに、短く、責めない表現に統一する
- `入力が間違っています` のような抽象文だけで終わらせず、どこを直せばよいかを添える
- 禁止語として、脅し・断定・人格評価に見える表現は使わない
- 例:
  - `テスト名は40文字以内で入力してください`
  - `科目名を入力してください`
  - `メモは200文字以内で入力してください`
  - `この入力はそのまま使えないため、内容を確認してください`
