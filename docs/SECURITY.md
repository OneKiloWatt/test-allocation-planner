# SECURITY

## 基本方針
- 未成年の学習データを扱う前提で、安全側に倒す
- 取らなくていいデータは取らない
- 断定評価、比較、共有を初期状態でやらない

## 扱うデータ
- テスト名
- テスト日程
- 教科
- 教科別目標点数
- 教科別点数
- 教科別勉強時間
- ブロック日

## 持たないデータ
- 学校名
- 住所
- 順位
- 友人情報
- 保護者情報
- 他人比較用データ

## 危険な仕様
- `このままだと危険`
- `この配分なら80点取れる`
- 他ユーザー比較
- ランキング
- 保護者/学校/塾への自動共有
- 広告目的の外部送信

## MVP時点の安全要件
- 通信は暗号化
- 認証ありの場合は最低限のアクセス制御
- データ削除導線を持つ
- 利用目的を明示する
- 提案文言は `参考プラン` `おすすめ配分` に留める

## データ保持方針
- テスト記録はユーザーが削除可能
- 退会または初期化で削除可能
- 長期保持を前提にしない

## 表現ルール
- 使う:
  - `前回より数学の配分を増やす案`
  - `目安`
  - `参考プラン`
- 使わない:
  - `勉強不足`
  - `志望校は無理`
  - `平均より低い`
